# Prueba para Konecta Colombia

Prueba hecha para Konecta en laravel 9, php 8.1 y mysql.

## Instalar

1. Instalar el proyecto: <br />
   Usando composer.

```bash
composer install
```

2. Correr migraciones:

```bash
php artisan migrate:fresh
```

3. Insertar Seeders:

```bash
php artisan db:seed
```
